# 회원가입 & 로그인

MOA와 파트너십 업무 협약을 완료한 후, 아래 단계를 따라 계정을 생성하고 로그인하세요.

## 회원가입 절차

<StepGuide>
  <Step number={1} title="전화번호 인증">
    휴대폰 번호를 입력하고 인증번호를 받아 본인 확인을 진행합니다.

    ![전화번호 인증 화면](/images/signup-step1.png)
  </Step>
  <Step number={2} title="계정 정보 입력">
    이름, 이메일 등 기본 계정 정보를 입력합니다.
  </Step>
  <Step number={3} title="소속 부동산 인증">
    사전에 발급된 **파트너사 전용코드**를 입력하여 소속 인증을 완료합니다.
    같은 중개법인/사무소의 여러 직원이 개별 계정으로 가입하고, 하나의 소속으로 묶이는 구조입니다.

    ![소속 부동산 인증 화면](/images/signup-step3.png)
  </Step>
  <Step number={4} title="로그인">
    가입이 완료되면 전화번호와 비밀번호로 로그인할 수 있습니다.
  </Step>
</StepGuide>

<Callout variant="tip" title="파트너사 전용코드">
  전용코드는 MOA와 파트너십 계약 시 발급됩니다. 코드를 받지 못하셨다면 담당자에게 문의하세요.
</Callout>

<Callout variant="info" title="복수 직원 가입">
  하나의 파트너사에 여러 직원이 개별 계정으로 가입할 수 있습니다. 각 직원은 동일한 파트너사 전용코드를 사용하여 가입하면 자동으로 같은 소속으로 연결됩니다.
</Callout>
