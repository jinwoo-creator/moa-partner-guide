# 매물 제안 요청 관리

MOA Partners의 메인 화면에서는 MOA가 발굴한 고객의 매물 제안 요청을 확인하고 참여할 수 있습니다.

## 3개 탭으로 관리하기

메인 화면은 3개의 탭으로 구성되어 있습니다:

![매물 제안 요청 3개 탭 화면](/images/customer-request-3tab.png)

### 매물 제안 요청
새로 등록된 매물 제안 요청을 확인할 수 있습니다. 고객의 주요 조건(위치, 면적, 예산 등)을 검토한 후 참여 여부를 결정하세요.

### 참여 중인 요청
현재 참여하고 있는 고객의 요청 목록입니다. 각 요청의 진행 현황을 확인하고 매물을 제안할 수 있습니다.

### 미참여·종료된 요청
참여하지 않았거나 종료된 요청을 확인할 수 있습니다.

## 참여 방법

![참여 방법 화면](/images/customer-request-howtoparticipate.png)

<StepGuide>
  <Step number={1} title="알림 수신">
    새로운 매물 제안 요청이 등록되면 파트너사 소속 전 직원에게 **카카오 알림톡**이 전송됩니다.
  </Step>
  <Step number={2} title="요청 검토">
    고객의 주요 조건(위치, 면적, 예산, 입주 시기 등)을 확인합니다.
  </Step>
  <Step number={3} title="참여 버튼 클릭">
    소속 직원 중 한 분이라도 참여를 클릭하면 해당 직원이 담당자로 참여됩니다.
  </Step>
</StepGuide>

<Callout variant="warning" title="3시간 내 참여 필수">
  매물 제안 요청은 **알림 수신 후 3시간 이내**에 참여해야 합니다. 마감 이후에는 자동으로 마감 처리되며 재참여가 불가능합니다.
</Callout>

<Callout variant="tip" title="중도 참여도 가능">
  기존에 진행 중이던 딜에서 파트너사가 이탈한 경우, 중도에 추가 참여도 가능합니다.
</Callout>
