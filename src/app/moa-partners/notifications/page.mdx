# 알림 내역

MOA Partners에서 발생하는 주요 알림을 확인하고 관리할 수 있습니다.

## 알림 패널 접근

화면 상단의 **알림 아이콘(종 모양)**을 클릭하면 알림 패널이 열립니다.

## 알림 종류

- **새로운 매물 제안 요청** — 새 고객의 매물 제안 요청이 등록됨
- **고객 조건 변경** — 참여 중인 고객의 주요 조건이 변경됨
- **진행 현황 업데이트** — 딜 진행 상태가 변경됨
- **메시지 수신** — MOA 담당자로부터 새 메시지 수신

## 필터 기능

알림 패널에서 다음 필터를 사용할 수 있습니다:

- **전체** — 모든 알림 보기
- **안읽음** — 아직 확인하지 않은 알림만 보기

## 읽음 처리

알림을 클릭하면 자동으로 읽음 처리되며, 관련 페이지로 이동합니다.

## 카카오 알림톡 연동

<Callout variant="info" title="카카오 알림톡">
  중요한 알림(신규 매물 제안 요청, 고객 조건 변경 등)은 MOA Partners 알림과 함께 **카카오 알림톡**으로도 전송됩니다. 별도의 설정 없이 가입 시 등록한 전화번호로 자동 발송됩니다.
</Callout>
